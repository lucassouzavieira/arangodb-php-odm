Graph

class Graph implements JsonSerializable (View source)

Represents an ArangoDB Graph.

Properties

protected string $id The internal id of this graph.
protected string $key The name of graph.
protected bool $isNew If this graph is a new one or a representation of existing document.
protected string $name Graph name.
protected bool $isSmart Flag if the graph is a smart graph.
protected string $revision The revision of graph.
protected int $numberOfShards Number of shards created for every new collection in the graph.
protected int $replicationFactor The replication factor used for every new collection in the graph.
protected int $minReplicationFactor The minimal replication factor used for every new collection in the graph.
If one shard has less than minReplicationFactor copies, we cannot write to this shard, but to all others.
protected ArrayList $edgeDefinitions An array of definitions for the relations of graph.
protected array $orphanCollections An array of additional vertex collections.
Documents within these collections do not have edges within this graph.
protected Database $database Database object of graph

bool addEdge(string $collection, array $attributes = [], bool $waitForSync = true, bool $returnNew = false)

Creates a new edge in the collection.
Within the attributes the edge has to contain a _from and _to value referencing to valid vertices in the graph.

Furthermore the edge has to be valid in the definition of the used.

Parameters

string $collection The name of the edge collection the edge belongs to.
array $attributes The object attributes to be stored. Must contains '_to' and '_from' keys.
bool $waitForSync Define if the request should wait until synced to disk. Default is true.
bool $returnNew Define if the response should contain the complete new version of the document. Default is false.

Return Value

bool A Edge object if edge exists. False if no graph with this name could be found
or this collection is not part of the graph or one of the vertices ('_to' or '_from') does not exist.
