AQLFunction

class AQLFunction implements EntityInterface (View source)

Represents an user defined aql function on server.

Properties

protected string $name Fully qualified name of user function.
protected string $code String representation of function body.
protected bool $isDeterministic An optional boolean value to indicate whether the function results are fully deterministic.
(function return value solely depends on the input value and return value is the same for repeated calls with same input)
protected bool $isNew If the entity is not an representation of a existing user function on server, this property is true.
protected Connection $connection Connection object.
protected array $deletion Stores the deletion data for an AQLFunction object.
